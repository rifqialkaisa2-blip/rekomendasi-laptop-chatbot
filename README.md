## Penjelasan Arsitektur (Sesuai Diagram Dosen)
Sistem ini mengimplementasikan alur chatbot sebagai berikut:
1. **User Input:** Pertanyaan pengguna dimasukkan melalui terminal.
2. **Bot Engine (NLP):** Menggunakan API Groq (Llama 3.3) untuk memproses bahasa alami.
3. **Knowledge Base:** Data spesifikasi laptop yang disimpan dalam variabel `laptop_data`.
4. **Response:** Jawaban cerdas yang dihasilkan oleh AI berdasarkan data laptop.

## Cara Menjalankan
1. Instal library: `pip install groq`
2. Jalankan: `python chatbot_rifqiial.py`
